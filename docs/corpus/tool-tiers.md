# Tool Tiers for Furniture Making

A progressive tool investment guide. Each tier unlocks new joints and project complexity. Always start at Tier 0 and add tools as projects demand them.

---

## Tier 0 — Ana White Minimum (~$200)

**Tools:** Tape measure, speed square, clamps (4+ bar clamps), pencil, drill/driver, circular saw

**Optional add-on:** Kreg pocket hole jig (~$50)

**Available joints:**
- Butt joint + screws
- Pocket screw joint (with Kreg jig)

**Cannot cut:** Dados, rabbets, mortises, dovetails, finger joints, half-laps

**Strengths:** Fast builds, forgiving of imprecision, paint-grade projects. Ana White's entire catalog lives here.

**Limitations:** Joints rely on mechanical fasteners. Visible screws unless filled and painted. Limited to right-angle construction.

**First project:** Simple shelf or workbench from 2x4s and plywood. Use pocket screws for face frames, butt joints with screws for structure.

---

## Tier 1 — Add a Router (~$100-150 more)

**Tools:** Everything in Tier 0 + plunge or fixed-base router, straight bit, edge guide, roundover bit

**Available joints (new):**
- Dado
- Rabbet
- Groove (for panel capture)
- Sliding dovetail (with dovetail bit + guide)

**Edge profiles (new):**
- Roundover
- Chamfer
- Ogee

**Still cannot cut:** Mortise and tenon (without a dedicated jig), hand-cut dovetails, precise miters, finger/box joints

**Strengths:** Shelves that interlock without fasteners. Panels captured in grooves instead of glued flat. Edge profiles elevate visual quality immediately.

**Limitations:** Freehand routing requires practice. Cuts are only as accurate as the edge guide or jig. Dust collection becomes important.

**First project:** Bookshelf with dado-captured shelves. Dados hold the shelves square and distribute load across the full width — a massive upgrade from pocket screws into end grain.

---

## Tier 2 — Add a Table Saw (~$300-500 more)

**Tools:** Everything in Tier 1 + table saw, dado stack, miter gauge, crosscut sled (shop-made)

**Available joints (new):**
- Finger/box joint (with a simple shop-made jig on the miter gauge)
- Half-lap
- Bridle joint
- Precise miters (with miter gauge or sled)
- Tenon cheeks (cut on table saw; pair with router mortise for full M&T)

**Approaching:**
- Mortise and tenon — router cuts the mortise, table saw cuts the tenon. First reliable M&T without hand tools.
- Splined miters — table saw cuts the miter and the spline slot

**Strengths:** Repeatable cuts. Rip lumber to exact width. Dado stack cuts flat-bottomed grooves faster than a router. Crosscut sled makes every cut square.

**Limitations:** Requires space. Blade changes for dado stack are slow. Good dust collection is now mandatory. Learning curve for safe operation.

**First project:** Table with half-lap stretchers. Half-laps are forgiving (fit is adjustable with shims or a few more passes) and the table saw makes them repeatable.

---

## Tier 3 — Full Shop (~$2000+ total)

**Tools:** Everything in Tier 2 + bandsaw, drill press, thickness planer, jointer

**Available joints (new):**
- All joints from lower tiers, executed with more precision
- Hand-cut dovetails (bandsaw for rough cuts, chisels for cleanup)
- Compound-angle joinery (drill press + bandsaw)
- Angled mortise and tenon
- Curved components (bandsaw)
- Domino/floating tenon (with Festool Domino, ~$1000 add-on)

**Strengths:** Mill rough lumber to exact dimensions (planer + jointer). Resaw thick stock into thinner boards (bandsaw). Every joint type is now achievable.

**Limitations:** Shop space, dust collection infrastructure, and learning time. Diminishing returns on joint quality — the jump from Tier 1 to Tier 2 matters more than Tier 2 to Tier 3 for most projects.

**First project:** Chair with angled mortise and tenon. Chairs demand precision in angled joinery, and Tier 3 tools finally make that achievable without heroic effort.

## Joint-to-Tool Feasibility Matrix

### How to read this matrix

When the skill recommends a joint, check the user's declared tool tier. If the joint shows "No" at their tier, suggest the nearest available alternative. For example:
- User at Tier 0 wants shelves: recommend butt + pocket screw (not dado)
- User at Tier 1 wants a table: recommend dado or rabbet for apron-to-leg (not half-lap)
- User at Tier 2 wants decorative box corners: recommend finger/box joint with jig (not dovetail)

### Upgrade path advice

Only recommend upgrading tiers when a project genuinely requires it. A pocket-screw bookshelf painted white is a perfectly valid bookshelf. Tool acquisition should follow project need, not aspiration.

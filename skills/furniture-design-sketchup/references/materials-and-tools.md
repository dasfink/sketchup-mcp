**Always model actual dimensions.** Standard lengths: 6, 8, 10, 12, 14, 16 ft.

Board feet: `BF = (T" × W" × L") / 144`

## Plywood

| Nominal | Actual | Notes |
|---|---|---|
| 1/4" | 0.250" | Usually true 1/4" |
| 1/2" | 0.500" or 15/32" (0.469") | Varies by supplier |
| 3/4" | 23/32" (0.719") | Standard big-box hardwood ply |
| 3/4" | 0.750" | Baltic birch, premium cabinet ply |

**Critical:** Standard 3/4" ply is ~23/32". A 3/4" dado will be loose. Always measure actual thickness for dado sizing.

**3/8" plywood is NOT commonly stocked** in hardwood species at big-box stores. Use 1/4", 1/2", or 3/4".

Standard sheet: 4' × 8'. Baltic birch: 5' × 5'.

### Grades

| Grade | Description |
|---|---|
| A-A | Both faces furniture-quality |
| A-C | One good face — standard for cabinet boxes |
| C-D | Construction/sheathing — not for finished work |

## Wood Species

### Indoor

| Species | Hardness (lbf) | Stain | Best for |
|---|---|---|---|
| Pine (Eastern White) | 380 | Poor — blotchy, needs conditioner | Painted furniture, beginner projects |
| Poplar | 540 | Poor — best painted | Painted furniture (pro choice) |
| Red Oak | 1,290 | Excellent | Stained furniture, most forgiving hardwood |
| Hard Maple | 1,450 | Difficult — use gel/dye | Workbenches, butcher block |
| Cherry | 950 | Good — darkens dramatically | Heirloom pieces, ages beautifully |
| Black Walnut | 1,010 | Excellent — barely needs stain | Show pieces, natural/oil finish |

### Outdoor

| Species | Rot Resistance | Notes |
|---|---|---|
| Pressure-Treated Pine | High | Ground contact OK. Use stainless/HDG fasteners. |
| Western Red Cedar | High | Lightweight, silvers to gray. |
| Redwood | High | Regional (West Coast). |
| Douglas Fir | Low–Moderate | Strong, above-grade only. |

**Fastener rule:** PT lumber requires hot-dipped galvanized or stainless steel — standard galvanized corrodes from the copper preservatives.

### Material-to-Finish Pairing

| Finish | Use these species | Don't use |
|---|---|---|
| Paint | Pine, poplar, SPF, birch ply | Walnut, cherry (wasted grain) |
| Stain | Red oak, poplar (dark stains), walnut (skip stain) | Pine without conditioner |
| Natural/oil | Walnut, cherry, white oak, maple | Construction-grade SPF |
| Outdoor | Cedar, redwood, PT lumber, white oak | Untreated softwood |

## Tool Tiers

### Tier 0 — Ana White Minimum (~$200)

Tape measure, speed square, clamps, pencil, drill/driver, circular saw. Optional: Kreg pocket hole jig (~$50).

**Joints:** Butt + screw, pocket screw.

### Tier 1 — Add a Router (~$100–150 more)

Plunge or fixed-base router, straight bit, edge guide, roundover bit.

**New joints:** Dado, rabbet, groove, sliding dovetail (with jig).

### Tier 2 — Add a Table Saw (~$300–500 more)

Table saw, dado stack, miter gauge, crosscut sled (shop-made).

**New joints:** Finger/box joint (jig), half-lap, bridle, precise miters, M&T (router mortise + table saw tenon).

### Tier 3 — Full Shop (~$2000+ total)

Bandsaw, drill press, thickness planer, jointer.

**New joints:** All joints with more precision. Hand-cut dovetails, compound-angle joinery, angled M&T, curved components.

| Joint | T0 | T1 | T2 | T3 |

| Butt + screw | Yes | Yes | Yes | Yes |
| Pocket screw | Jig | Jig | Jig | Jig |
| Dado | No | Yes | Yes | Yes |
| Rabbet | No | Yes | Yes | Yes |
| Groove | No | Yes | Yes | Yes |

| Finger/box joint | No | No | Jig | Jig |

**Upgrade rule:** Only recommend upgrading tiers when a project genuinely requires it. A pocket-screw bookshelf painted white is a perfectly valid bookshelf.

## Finish Workflows

### Paint (most forgiving — best for Tier 0)

1. Fill holes with wood filler
2. Sand 120–150 grit
3. Clean (vacuum + damp cloth)
4. Prime (always — use stain-blocking primer on knotty pine)
5. Paint with foam roller, 3–4 thin coats, sand 220 between

### Stain (less forgiving — gaps and filler show)

1. Test on scrap (not optional)
2. Sand to 150–180 grit
3. Pre-stain conditioner on softwoods
4. Apply stain, wipe excess before tacky
5. Polyurethane topcoat (stain alone has zero durability)

### Natural/Oil (shows every flaw — Tier 2–3 builds)

1. Sand progressively to 220 grit
2. Raise grain (damp cloth, dry, sand 220)
3. Flood with oil (Danish, tung, or BLO), wait 15–30 min
4. Wipe off ALL excess
5. 2–3 coats, 24h between, buff final coat

### Outdoor

1. Exterior-rated products only
2. Penetrating stain/sealer (not film-forming)
3. Reapply every 1–2 years
4. Or leave unfinished (cedar/white oak silver naturally)

### Finish → Joint Decision Rule

| Finish | Joint implication |
|---|---|
| Paint | Simplest joints fine — pocket holes fillable, butt joints disappear |
| Stain | Tighter joints needed — wood filler won't take stain. Pocket holes need face-grain plugs |
| Natural/oil | Cleanest joints — dovetails and M&T become visual features |
| Outdoor | Mechanical fasteners preferred — glue joints may fail from moisture cycling |

**If the user says "I'm painting it," do not recommend dovetails.**
